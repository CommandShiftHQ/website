## Expected Behaviour

When `booleanToWord` is passed `true`, it should return `'Yes'`. When it is passed `false`, it should return `'No'`.

## Assertion

- returns Yes when passed true

- returns No when passed false

## Run tests

```bash

npm run test -- booleanToWord

```
