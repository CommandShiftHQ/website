## Expected Behaviour

When `numberToReversedDigits` is passed a number, it should return a _reversed_ array of that number's digits.

E.g.

`12345`

becomes

`[5, 4, 3, 2, 1]`

## Assertion

- returns a reversed array of the number's digits

## Run tests

```bash

npm run test -- numberToReversedDigits

```
