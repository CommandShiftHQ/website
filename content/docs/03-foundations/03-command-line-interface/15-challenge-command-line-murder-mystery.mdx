Welcome Developer. Wait, did I say Developer? I meant Detective. Let’s mix both hats.

There's been a murder in Terminal City, and TCPD needs your help.

**Very Important:**
Don't use a text editor.
The objective of this exercise is to test all you’ve learn so far, while getting more comfortable using the command line.
You will find that it is an incredibly powerful tool on its own, if you haven’t already.
You can, however, use Google, course materials, cheatsheets or any other materials available.

## Step 1: Set It Up

1. Move to your projects folder, which should live in your home directory.
2. Use the following command: `git clone https://github.com/MCRcodes/cli-mystery.git`.

The last step will clone the project which contains a couple of files needed for this challenge onto your own machine.

## Step 2: Investigate

There are three directories in this repository - **mystery**, **steps** and **hints**.

- `mystery` contains all of the information you will need to go through to solve the mystery.
- `steps` contains some instructions to guide you on your journey.
- `hints` contains some hints for some of the steps, which you should use as a last resort, after attempting without them. Not every step has a hint.

To get started, open a Terminal, go to the location of the files, and start by reading the file `steps/step0`.

Have fun, remember that getting stuck is half of it and it will happen frequently as you go. So, push through and you’ll get there.

Enjoy the journey instead of focusing so much on the end goal.

Let us know how you’re getting along!
