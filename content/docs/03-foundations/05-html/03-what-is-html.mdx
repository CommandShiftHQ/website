**HTML** stands for **HyperText Markup Language**.

# Evolution of HTML

Since the web was first created, there have been several different versions of HTML, each of them an improvement of the last.

The first HTML version was released in 1991 by **Tim Berners-Lee** as a way for CERN researches to share documents with each other. It all started [here](http://info.cern.ch/).

HTML 2.0 was out by 1995, 3.2 in 1997, 4 in 1997 and 5, the most current one, in 2014.
